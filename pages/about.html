About the System

KatapatanEagerGNC LMS

A Student Record Management System built with vanilla JavaScript using Object-Oriented Programming principles, jQuery, and localStorage for persistent data — no backend required.

Students
View grades and academic standing per subject.
Teachers
Add subjects and input grades per student per period.
Admins
Manage all users, sections, and export reports.
